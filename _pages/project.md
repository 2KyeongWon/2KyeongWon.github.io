## 재고 관리 3인 프로젝트입니다.

#### 제작기간 21년4월10일~ 21년4월20일

> 개발 목적

핸드폰으로 손 쉽게 재고관리가 가능한 웹사이트
오늘 사용한 재고를 출고하고 주문을 통해 입고를 하는 간단한 프로젝트 입니다.

> 개발 순서

업무분석 -> 인터페이스 구성 -> 시스템제작 -> DB구축 -> 오류수정 -> 완성

1. 업무 분석
   주제 선정
2. 인터페이스 구성
   기본 템플릿 AdminLTE를 이용
3. 시스템 제작
   Spring, ajax, jQuery를 사용하여 제작
4. DB 구축
   OracleDB 사용
5. 오류 수정
